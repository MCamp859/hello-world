Reference Implementations
=========================


By following this guide, you tried a few commands to familiarize
yourself with the features of the edgesoftware CLI.


To run your first inference application using Edge Insights for Fleet,
try one of these reference implementations:


-  `Vehicle Event
   Recording <http://software.intel.com/content/www/us/en/developer/articles/reference-implementation/vehicle-event-recording.html>`__


-  `Driver Behavior
   Analytics <https://software.intel.com/content/www/us/en/developer/articles/reference-implementation/driver-behavior-analytics.html>`__


-  `Cargo
   Management <https://www.intel.com/content/www/us/en/developer/articles/reference-implementation/cargo-management.html>`__


-  `Public Transit
   Analytics <https://software.intel.com/content/www/us/en/developer/articles/reference-implementation/public-transit-analytics.html>`__


-  `Automated License Plate
   Recognition <https://www.intel.com/content/www/us/en/developer/articles/reference-implementation/automated-license-plate-recognition.html>`__


-  `Address Recognition and
   Analytics <https://www.intel.com/content/www/us/en/developer/articles/reference-implementation/address-recognition-and-analytics.html>`__


-  `Work Zone
   Analytics <https://www.intel.com/content/www/us/en/developer/articles/reference-implementation/work-zone-analytics.html>`__
